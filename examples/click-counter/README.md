# Click Counter
This application demonstrates practical use of [MarqueeTitle.pause()](https://git.io/vFVpC), [MarqueeTitle.resume()](https://git.io/vFVpn), and [MarqueeTitle.start()](https://git.io/vFVpZ).

# Get Started
Go to <https://cameronsamuels.com/marqueetitle/examples/click-counter> to interact with the example.

Also, you can view the documentation of the methods used in this example including [MarqueeTitle.pause()](https://git.io/vFVpC), [MarqueeTitle.resume()](https://git.io/vFVpn), and [MarqueeTitle.start()](https://git.io/vFVpZ).
